Projects

C4G BLIS

The Computing 4 Good Basic Laboratory Information System is a web-based laboratory management system. Currently deployed in several countries, including Tanzania and Cameroon, the development of C4G BLIS focuses on a model based on direct feedback and feature requests from the end-users' laboratories. The platform uses a mobile WAMP stack, using MySQL and PHP.
